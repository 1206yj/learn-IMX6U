# Ubuntu用户与用户组

## 用户

![20200414_062735_52](image/20200414_062735_52.png)

![20200414_063016_19](image/20200414_063016_19.png)

* 普通用户 （UID 500-60000）
* 伪（系统）用户（UID 1-499）
* root (UID = 0)
* /etc/passwd /etc/shadow

* 想要创建用户，不用sudo？？？怎么想的
* 凡是操作非本组，分本用户的账户管理内容均需要sudo


## 用户组

* 分组管理
* 用户组存储在/etc/group
